# Antlr Editor

See example into deployed [Storybook](https://nicolaval.github.io/antlr-editor/index.html)

## Usage

```bash
yarn add antlr-editor
```

| Name            | Type     | Default value |

| script          | string   | -             |

| languageVersion | string   | -             |

| suggesterURL    | string[] | -             |
| tools           | Tools \* | -             |
